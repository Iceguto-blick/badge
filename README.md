## Comandos Git

- git clone (clonar um repositorio)
- git branch (mostrar a branch atual)
- git checkout -b (criar uma nova branch)
- git status (mostra qual arquivo foi mudado por fora do git)
- git add . (adiciona todos arquivos do path no stage changes)
- git add arquivo (adiociona um unico arquivo no stage changes)
- git commit -m (salvar com comentario)
- git push origin "nome do arquivo" (manda os arquivos do repositorio local para o github nuvem)
